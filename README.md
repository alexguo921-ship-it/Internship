# AI内部竞聘评审系统 · 运行说明

## 快速启动

```bash
# 1. 进入目录
cd 腾讯作业

# 2. 安装依赖（建议 Python 3.9+）
pip install -r requirements.txt

# 3. 启动 Demo
streamlit run app.py
```

浏览器会自动打开 `http://localhost:8501`

---

## 功能模块

| 标签页 | 演示内容 |
|--------|---------|
| 📊 总览仪表板 | 360人评分分布、区间占比、部门看板、两周甘特图 |
| 👥 候选人列表 | 可筛选/搜索的评审表格、个人雷达图 |
| 🔍 人工复核 | 模拟 HR 对黄区候选人进行审批操作 |
| 📋 能力发展报告 | 一键生成个性化报告（支持 Claude API 或智能模板） |
| 📈 效果评估 | KPI 仪表盘、AI赋能前后对比、离职率逻辑路径 |

---

## Claude API 配置（可选）

在左侧边栏「Claude API」栏输入 API Key，即可使用真实 AI 生成能力报告。

未配置时自动使用智能模板报告，同样可完整演示功能。

---

## 演示数据说明

- 系统内置 360 名候选人的模拟数据（固定随机种子，每次运行结果一致）
- 评分权重、复核阈值可在左侧边栏实时调整，图表自动更新
